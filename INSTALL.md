# 📱 How to Install Zeeshan Tracker on Your Phone

## iPhone (Safari only)
1. Open `index.html` in **Safari**
2. Tap the **Share** button (box with arrow icon at the bottom)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **"Add"** ✅

## Android (Chrome)
1. Open `index.html` in **Chrome**
2. Tap the **3-dot menu** (top right)
3. Tap **"Add to Home Screen"** or **"Install App"**
4. Tap **"Install"** ✅

## 🌐 Best Way — Host It Free Online

### Netlify (easiest, recommended):
1. Go to https://netlify.com → free sign up
2. Drag the entire `zeeshan-tracker-green` folder onto Netlify
3. You'll get a link like `https://zeeshan-tracker.netlify.app`
4. Open that link on any phone → follow steps above to install ✅
